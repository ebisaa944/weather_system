🌤️ Weather API Documentation

Base URL: /api/v1/

Current Weather

GET /weather/current/

Get current weather conditions for a city.

Parameters

city (required) - City name (e.g., London, Tokyo)
lat (optional) - Latitude for coordinate-based search
lon (optional) - Longitude for coordinate-based search

Example Request

GET /api/v1/weather/current/?city=London

Example Response

{
    "success": true,
    "data": {
        "city": "London",
        "country": "GB",
        "temperature": {
            "current": 15,
            "feels_like": 14,
            "min": 13,
            "max": 17
        },
        "condition": {
            "main": "Clouds",
            "description": "Scattered clouds",
            "icon": "03d"
        }
    }
}

Weather Forecast

GET /weather/forecast/

Get 5-day weather forecast for a city.

Parameters

city (required) - City name
days (optional) - Number of days (1-7, default: 5)

City Search

GET /weather/search/

Search for cities by name (autocomplete).

Parameters

q (required) - Search query (minimum 2 characters)

Air Quality

GET /weather/air-quality/

Get air quality data by coordinates.

Parameters

lat (required) - Latitude
lon (required) - Longitude
